## Key Gaps & Risks
- 12 broken frontend calls: admin CRUD (PUT/DELETE for vocab, phrases, reading, writing + stats + GET)
- Admin endpoints lack role-based security (@PreAuthorize/@Secured missing)
- Missing ownership validation on conversations, reading passages, exercise submissions
- PUT /api/profiles/{id}/levels and DELETE /api/profiles/{id} missing @RequireProfileOwnership
- Frontend-backend path mismatches: module-progress, tutor/conversations, content/vocab, assessments
- New block advance/exercises endpoints need frontend integration
- Session completion now requires all exercises completed — frontend must adapt

## Active Development
- Daily exercises restructure: block-level exercise tracking, advance validation, enriched session responses
- Learning Path & Mastery redesign: new aggregates (LearningPath, LearningUnit, VocabMastery), V9.x migrations
- Auth hardening: optimistic locking, ownership checks, JWT 60min/7d, 401 vs 403 correctness
