# S2GRUPO Brand Identity — V4

Corporate identity reference for all S2GRUPO documents and presentations.

## Color Palette (Primary)

| Name | Token | Hex | Usage |
|------|-------|-----|-------|
| Azul principal | `BLUE` | `#0545b0` | Primary brand, titles, backgrounds |
| Azul claro | `BLUE_LIGHT` | `#5a84ce` | Secondary accent, highlights |
| Lima | `LIME` | `#dbf266` | Accent on solid backgrounds only |
| Crudo | `CREAM` | `#ededeb` | Light backgrounds, cards |
| Negro | `BLACK` | `#000000` | Body text |
| Blanco | `WHITE` | `#ffffff` | Text on blue, white backgrounds |

## Typography

| Role | Font | Notes |
|------|------|-------|
| Principal | **Manrope** | Corporate identity (Google Fonts) |
| Fallback Office | **Arial** | PPTX, DOCX, and any Office document |
| Code | **Consolas** | Code blocks in presentations |

> In PPTX and Office documents, always use **Arial** as the text font.

## Writing Rules

- Always write **S2GRUPO** in uppercase, no space between S2 and GRUPO.
- Claim: **"Anticipando un mundo ciberseguro"**
- Never abbreviate or alter the company name.

## Logo Versions

| Version | Use |
|---------|-----|
| Logo + claim | Default for all documents |
| Logo without claim | When space is limited (min width: 2 cm) |
| Symbol only | Social media avatars or when name is adjacent |

### Safety Area

Defined by the x-height of the R in the logo. No element may enter this zone.

### Minimum Sizes

- With claim: **3.5 cm** wide (print)
- Without claim: **2 cm** wide (print)

## Color Combinations

**Correct:** Azul on blanco, Blanco on azul, Negro on crudo, Azul on crudo.

**Incorrect:** Lima on blanco (legibility), Azul on negro, Non-corporate colors.

## Quick Rules

1. Primary blue is `#0545b0` — never use the old `#003399`.
2. Lima (`#dbf266`) only on solid backgrounds, never on white.
3. Crudo (`#ededeb`) replaces light grays for backgrounds.
4. For full palette (gradients, semaphoric), see [references/brand-guide.md](references/brand-guide.md).
5. For PPTX design tokens, see skill `s2-docs:pptx`.
